Sinatra is a micro-framework (150KB) and a domain-specific language for easy development of RESTful web applications in Ruby. In fact, this easy:

# application.rb
require 'rubygems'
require 'sinatra'
get '/' do
  'Hello world!'
end

Get on stage

$ sudo gem install sinatra
$ ruby application.rb
